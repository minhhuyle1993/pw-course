# Key takeaways lession 02
## Git
### Các điểm chính về git:
- Git có 3 states (working, staging, repository)
- Undo (restore, reset)
- Làm việc trên nhiều máy (pull, push, remote)
- Làm việc với nhiều người (branching)
- Xử lý lỗi (resolve conflict, rebase, squash)

### Commit convention:
"type": "description"

**Type**
- chore: sửa nhỏ lẻ, chính tả, thêm file
- feat: thêm tính năng mới, test case mới
- fix: sửa lỗi 1 test trước đó 

### Basic workflow:
init -> Config (nếu chưa có) -> add -> commit -> push
